Meeting-to-Action Prototype

This repository contains a JavaScript prototype that parses a meeting transcript, extracts commitments, formats tasks, and drafts follow-up emails.

Run locally

npm install
npm test
npm start

Files

  • src/parser.js — transcript cleanup and speaker segmentation
  • src/extractor.js — commitment extraction and deadline recognition
  • src/formatter.js — task object formatting
  • src/email.js — follow-up email drafting
  • src/coordinator.js — pipeline review summary

Deployment

This site is published automatically via GitHub Pages from the docs/ folder.

Live demo: https://shreyam5.github.io/multi-agent-meeting-bot/
